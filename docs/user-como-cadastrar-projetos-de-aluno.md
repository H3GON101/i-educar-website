title: 17. Como Cadastrar Projetos de Aluno?
---

<div class="textoJustificado">

Para realizar o cadastramento de áreas do conhecimento, acesse o **Módulo Escola >> Cadastros >> Tipos >> Alunos >> Tipos de projetos>> Novo**.

</div>
